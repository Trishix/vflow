# VFLOW

VFLOW is a node-based AI workflow builder for reusable prompt, model, markdown, and annotation flows.

## What It Does

- Build workflows with drag-and-drop nodes
- Chain prompts and AI models together
- Keep workflows local in the browser
- Bring your own API keys
- Reuse workflows across tasks and projects

## Getting Started

Open the app, create a workflow, add nodes, and connect them together.

## Stack

- Next.js App Router
- React Flow
- Zustand
- Tailwind CSS
